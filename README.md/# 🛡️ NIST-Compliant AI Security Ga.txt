**A DevSecOps implementation of the NIST AI Risk Management Framework (RMF) using Cloudflare AI Gateway.**

## üìå Project Overview
Direct access to Large Language Models (LLMs) poses significant security risks, including data leakage, unmanaged costs, and Denial of Service (DoS) attacks.

This project implements a **Secure AI Proxy** that governs all traffic to Llama-3, enforcing the **CIA Triad** (Confidentiality, Integrity, Availability) before requests ever reach the model.

## üöÄ Architecture
**User/App** ‚û°Ô∏è **Cloudflare Gateway (The Shield)** ‚û°Ô∏è **Workers AI (Llama-3)**

### Security Controls Implemented
| Principle | Implementation | Result |
| :--- | :--- | :--- |
| **Confidentiality** | Payload Redaction | PII and secrets are never stored in persistent logs. |
| **Integrity** | API Authentication | Access restricted to authorized API tokens only. |
| **Availability** | Rate Limiting | Spammers are blocked (HTTP 429) after 3 requests/min. |
| **Observability** | NIST "Measure" | Full visibility into token usage, latency, and errors. |
